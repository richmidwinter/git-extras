git-root(1) -- Show repository root directory
===============================================

## SYNOPSIS

`git-root`

## DESCRIPTION

Shows the root directory of the repository.

## EXAMPLES

  Outputs a repo root directory:

    $ git root

        /home/username/repos/myrepo
